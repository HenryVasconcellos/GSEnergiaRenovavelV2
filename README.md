# GSEnergiaRenovavelV2


Simulação de Sistema IoT para Gestão de Energia em Escritório (Climatização, Iluminação e Equipamentos



Este projeto simula o impacto de um sistema IoT de automação de energia em um escritório, atuando em climatização, iluminação e equipamentos, a partir de dados de consumo por zona com resolução de 15 minutos.
